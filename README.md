
# Fleet Management System | نظام إدارة الأسطول 🚚💨

A Python project using **OOP** to manage logistics, vehicles, and drivers.
مشروع بلغة بايثون يستخدم **البرمجة الكائنية** لإدارة العمليات اللوجستية، المركبات، والسائقين.

---

## 🚀 Main Features | المميزات الرئيسية

* **Fleet Control (إدارة الأسطول):** Register and track Drivers, Vehicles, and Shipments.
  *(تسجيل وتتبع السائقين، المركبات، والشحنات بسهولة).*
* **Smart Dispatch (الإرسال الذكي):** Matches shipments to trucks based on weight, fuel/battery, and driver license type.
  *(ربط الشحنات بالشاحنات تلقائياً حسب الوزن، الوقود/البطارية، ونوع رخصة السائق).*
* **Eco-Friendly Fleet (تنوع الشاحنات):** Supports both Gasoline and Electric trucks.
  *(يدعم الشاحنات التقليدية والكهربائية مع نظام تعبئة وشحن).*
* **Trip Completion (إنهاء الرحلات):** Updates shipment status to `delivered` and makes drivers available again.
  *(تحديث الشحنات إلى "تم التوصيل" وتفريغ الشاحنة وإتاحة السائق مجدداً).*

---

## 🛠️ Concepts Used | المفاهيم المستخدمة

* **Language:** Python 3.
* **OOP principles:** Inheritance (الوراثة) & Polymorphism (تعدد الأشكال).
* **Error Handling:** Prevention of overloading or assigning busy drivers using `ValueError`.
  *(منع الأخطاء مثل الوزن الزائد أو تعيين سائق مشغول باستخدام `ValueError`).*
